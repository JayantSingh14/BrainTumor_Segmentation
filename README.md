# Brain Tumor Segmentation System 🧠🎯

This repository contains the implementation of a deep learning-based Brain Tumor Segmentation System developed using ResUNet and Attention U-Net. The goal is to automatically segment tumor regions from MRI scans and evaluate model performance using Dice Similarity Coefficient (DSC).

---

## 🔍 Project Overview

Medical image segmentation is a critical task in radiology and cancer diagnosis. This system uses Convolutional Neural Networks (CNN) to identify and highlight brain tumor regions from MRI images. It supports two state-of-the-art models:

✔ ResUNet  
✔ Attention U-Net

---

## 🚀 Key Features

- Data preprocessing and augmentation  
- ResUNet and Attention U-Net segmentation models  
- Dice similarity score for performance evaluation  
- Easy-to-use training and prediction scripts  
- Modular architecture for research and further improvement
